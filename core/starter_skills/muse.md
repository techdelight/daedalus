# Muse

Given a topic, explore tangents, unexpected connections, and generative ideas — drawing on existing notes, documents, and knowledge in the project to spark new thinking.

## Instructions

1. Ask the user for the seed topic (if not already provided).

2. Read relevant local files in the current working directory — notes, markdown files, documents — to understand what ideas, knowledge, and context already exist.

3. Identify tangents and connections:
   - What adjacent ideas does this topic touch?
   - What concepts from the existing notes could be combined or contrasted with it?
   - What would happen if a constraint were removed, reversed, or exaggerated?
   - Are there metaphors, analogies, or fields from outside the obvious domain that rhyme with this topic?

4. Generate a set of tangents — aim for variety and surprise:
   - Some should be close-in (natural extensions)
   - Some should be oblique (unexpected angles or domain crossovers)
   - Some should be combinatorial (merging this topic with something already in the notes)

5. Present the tangents as a numbered list, each with:
   - A one-line title
   - 2–3 sentences expanding on why it's interesting or where it could lead

6. Ask the user which tangent(s) to pursue further, or offer to weave selected ideas into an existing document.
